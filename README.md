# it-is

Data types methods validator

```js

if ( is.String('Hola') ) // true
if ( is.String({}) ) // false

var arr = []

if ( is.String(arr) ) // false
```

### Avalible methods

```js
is.String()

is.Number()

is.Integer()

is.Floating()

is.Function()

is.Object()

is.Array()

is.Infinity()
```
